## 三、事务一致性检查（WAL 模式验证）

### 3.1 WAL 状态机

```
┌──────────────────────────────────────────────────────────────────────┐
│                         WAL 事务状态机                                │
├──────────────────────────────────────────────────────────────────────┤
│                                                                      │
│   1. PENDING ─────────────────────────────────────────────────────   │
│      │         INSERT OR REPLACE (symbol, timestamp) UNIQUE        │
│      ▼                                                              │
│   2. 业务执行                                                        │
│      │         signal → decision → order                            │
│      ▼                                                              │
│   ┌────────────────────────────────────────────────────────────┐    │
│   │ 3a. CONFIRMED  ←─── 成功                                    │    │
│   │     │                                                      │    │
│   │     └─── order_result = "OK"                              │    │
│   │                                                             │    │
│   │ 3b. FAILED    ←─── 失败                                     │    │
│   │     │                                                      │    │
│   │     └─── order_result = error_message                       │    │
│   └────────────────────────────────────────────────────────────┘    │
│                                                                      │
│   4. 崩溃恢复                                                        │
│      │         load_latest()                                      │
│      ▼                                                              │
│   ┌────────────────────────────────────────────────────────────┐    │
│   │ PENDING 状态保留，由上层（Trader）通过交易所 API 确认       │    │
│   │ ⚠️ 不再简单标记为 FAILED                                    │    │
│   └────────────────────────────────────────────────────────────┘    │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘
```
