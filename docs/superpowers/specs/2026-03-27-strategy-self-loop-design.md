一、核心设计
================================================================

一个品种 = 一个协程 = 一个 Trader

Engine 管 spawn/stop/监控/重启，Trader 自己 loop。

二、架构图
================================================================

┌─────────────────────────────────────────────────────────────┐
│  Engine                                                   │
│  ├── InstanceMap: HashMap<symbol, InstanceHandle>        │
│  ├── spawn(symbol)                                      │
│  ├── Monitor 协程                                       │
│  └── stop(symbol)                                       │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  h_15m::Trader.loop()                                   │
│  loop:                                                   │
│    record = TradeRecord.new()                           │
│    record.market = get_market()                        │
│    record.signal = signal_generator.generate()          │
│    record.check_*() = check()                          │
│    if all passed:                                       │
│        executor.send_order()                             │
│        executor.save_record()                           │
└─────────────────────────────────────────────────────────────┘

四、executor.rs 职责
================================================================

┌─────────────────────────────────────────────────────────────┐
│  Executor                                                │
├─────────────────────────────────────────────────────────────┤
│  send_order(signal) → 发送到交易所                       │
│  save_record(record) → 保存到 SQLite                    │
│  load_record(symbol) → 从 SQLite 恢复                   │
└─────────────────────────────────────────────────────────────┘

五、TradeRecord (唯一的一张表)
================================================================

┌─────────────────────────────────────────────────────────────┐
│  TradeRecord                                             │
├─────────────────────────────────────────────────────────────┤
│  基础信息 (symbol, timestamp, interval)                 │
│  行情快照 (price, volatility, market_status)            │
│  持仓快照 (交易所持仓 + 本地持仓记录)                    │
│  策略状态 (trader_status, signal, confidence)           │
│  账户状态 (available, used_margin, unrealized_pnl)     │
│  订单执行 (order_type, direction, quantity, result)       │
│  检查表 (check_*)                                        │
└─────────────────────────────────────────────────────────────┘

六、传递流程
================================================================

loop:
  1. TradeRecord.new()
  2. record.market = get_market()
  3. record.signal = signal_generator.generate()
  4. record.check_*() = check()
  5. if all passed:
       executor.send_order()
       executor.save_record()
  6. 等 interval
