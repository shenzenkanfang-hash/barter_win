指标逻辑文档

================================================================================
一、指标体系概述
================================================================================

指标分为两个层级：

1. 日线指标 - 3个核心，辅助决策
2. 分钟级指标 - 3个核心，用于交易决策

================================================================================
二、日线指标（辅助决策）
================================================================================

| 指标 | 用途 | 计算方式 |
|------|------|---------|
| 历史价格位置 | 判断当前价格在历史中的分位 | percentileofscore(全历史close, 当前close) |
| TR效率比 | 1%振幅对应时间 | 总天数 / 总振幅 |
| Pine颜色 | 趋势方向 | MACD + EMA10/20 + RSI |

说明：日线指标用于辅助判断市场整体状态，不直接触发交易

================================================================================
三、分钟级指标（核心交易）
================================================================================

| 指标 | 用途 | 增量计算 | 说明 |
|------|------|---------|------|
| TR | 波动率 + 突破过滤 | O(1) | 突破时TR放大=真突破，收缩=假突破 |
| Pine颜色 | 趋势信号 | O(1) | MACD + EMA10/20 + RSI |
| 价格位置 | 是否突破周期极值 | O(1) | (close-low)/(high-low)，事件驱动 |

指标计算优先级：
1. TR - 优先计算，用于判断是否值得关注
2. Pine颜色 - 次优先，趋势方向确认
3. 价格位置 - 最后计算，判断是否在极值

================================================================================
四、分钟级指标结构
================================================================================

```rust
struct MinuteIndicators {
    // TR - 波动率突破判断
    tr_hhv: MonotonicQueue,  // 20周期最高
    tr_llv: MonotonicQueue,  // 20周期最低
    tr_anchor: f64,          // 锚定价格

    // Pine颜色 - 趋势信号
    ema_fast: EMA,           // 12周期
    ema_slow: EMA,           // 26周期
    ema_signal: EMA,         // 9周期
    ema10: EMA,
    ema20: EMA,
    rsi_up: RMA,
    rsi_down: RMA,

    // 价格位置 - 周期极值判断
    price_history: RingBuffer,  // 20周期价格
}
```

================================================================================
五、Pine颜色判断逻辑
================================================================================

| 颜色 | MACD条件 | RSI条件 | 说明 |
|------|----------|---------|------|
| 纯绿 | MACD >= Signal && MACD >= 0 | - | 强势多头 |
| 淡绿 | MACD <= Signal && MACD >= 0 | - | 弱势多头 |
| 纯红 | MACD <= Signal && MACD <= 0 | - | 强势空头 |
| 淡红 | MACD >= Signal && MACD <= 0 | - | 弱势空头 |
| 紫色 | - | RSI >= 70 或 RSI <= 30 | 极值区域 |

================================================================================
六、TR指标说明
================================================================================

TR (True Range) 用于判断真突破vs假突破：
- 突破时TR放大 -> 真突破
- 突破时TR收缩 -> 假突破

计算：
- tr_hhv: 20周期最高价
- tr_llv: 20周期最低价
- tr_anchor: 锚定价格（开盘价或前一日收盘价）

================================================================================
七、价格位置说明
================================================================================

公式：(close - low) / (high - low)

用途：判断当前价格是否处于周期极值
- 接近1: 价格在周期高点
- 接近0: 价格在周期低点

驱动方式：事件驱动，实时更新

================================================================================
八、已删除的冗余指标
================================================================================

以下指标在分钟级交易中不再使用：

- Jerk (加加速度)
- Velocity (速度)
- Acceleration (加速度)
- Power (功率)
- 复杂多周期TR计算

原因：分钟级交易需要快速、简单的决策，这些指标增加复杂度但收益有限。
