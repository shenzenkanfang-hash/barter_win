## 10. 改进建议

### 10.1 短期优化 (1-2天)

1. **收紧 Pin Bar 参数**
   - 提高 body_ratio 阈值至 20%
   - 提高 wick_ratio 阈值至 70%
   - 减少无效信号

2. **添加趋势过滤**
   - 使用 EMA(20) 方向过滤
   - 只在趋势方向交易（趋势多 → 只做多；趋势空 → 只做空）

3. **添加固定止损**
   - 止损幅度：入场价的 0.5%
   - 止盈幅度：入场价的 1.0%

4. **减少交易频率**
   - 每日最大交易次数：10 次（当前无限制）
   - 过滤连续同向信号（5根K线内不重复入场）

### 10.2 中期优化 (1周)

1. **实现追踪止损**
2. **添加波动率过滤**（低波动时不交易）
3. **分批建仓**（避免一次性全仓）
4. **历史回测框架**（支持多组参数对比）
