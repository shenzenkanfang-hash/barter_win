## 四、架构说明

### 4.1 V1.1 流式回放架构

```
CSV Loader → KlineLoader (Iterator)
          → StreamTickGenerator (60 ticks/K-line)
          → MemoryInjector (SharedMemory)
          → ReplayController (speed control)
```

### 4.2 核心文件变更

| 文件 | 变更 | 说明 |
|------|------|------|
| kline_loader.rs | 重写 | Parquet → CSV 简化方案 |
| replay_controller.rs | 修复 | tokio::sleep → std::thread::sleep |
| noise.rs | 完善 | 添加 rand feature 支持 |
| memory_injector.rs | 完善 | Tick注入逻辑 |
| backtest/strategy.rs | 修复 | dec!宏运行时求值问题 |
| perf_test/engine_driver.rs | 修复 | 添加dec宏导入 |
| gateway/interceptor.rs | 修复 | 添加OrderStatus导入 |

### 4.3 Tick生成算法

```
牛市 K线（收盘 >= 开盘）: 路径 O → L → H → C
熊市 K线（收盘 < 开盘）: 路径 O → H → L → C

每段价格根据距离比例分配 Tick 数量：
dist[0] = |O-L|, dist[1] = |L-H|, dist[2] = |H-C|
total = sum(dist)
ticks_per_seg = dist / total * 60
```
